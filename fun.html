Mini Circuit Maker

Access Here

Simple circuit editor designed to create mini diagrams for the Node MCU microcontroller cause I was unfond of existing solutions

Features several component types, eg wires, resistors, and LEDS; but is very much a work in progress.

While it is possible to make simple diagrams, the application is still very much a work in progress.

Yet to add:

  • Erasing
  • Custom components
  • Fix issues (Eg, wires are able to go directly into the node mcu)
