Open Builder

Open builder is a 'Minecraft-like' game I made to learn about using networking to create a multiplayer game, and how to integrate a scripting language in a game.

This was created using C++, with OpenGl and Sfml for the graphics, ENet for reliable UDP sockets, and Lua/Sol2 for scripting support.

Github • YouTube • 'Demos'

Cellular Automaton

Collection of cellular automaton projects implemented using C++ and SFML for graphics.

Included are Conway's Game of Life, Predator and Prey, Wire World, Langton's Ant, and Brian's Brain
