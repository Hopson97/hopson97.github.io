Minecraft In One Week

August 2017

This was a project created for a challenge in the summer of 2017.

The challenge was to try and implement as much of the Minecraft game as I could in a week, and I recorded my proccess throughout. People continued to develop the project after I finished it, adding features such as a day/night cycle.

Features

  • Procederal World Generation
  • Biomes
  • 'Infinite' World
  • Break Blocks
  • Place Blocks
  • Collision Detection
  • Simple Inventory System
